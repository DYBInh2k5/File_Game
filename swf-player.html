SWF Player

Chạy các game Flash `.swf` trong thư mục hiện tại bằng `Ruffle`.

Đang chờ chọn game.
Trang này nên mở qua local server, ví dụ http://127.0.0.1:8000/swf-player.html.

Nếu game không chạy, nguyên nhân thường là file dùng ActionScript chưa được `Ruffle` hỗ trợ đầy đủ.

Chưa tải game Engine: Ruffle Web
Chọn một game ở khung bên trái rồi bấm chạy.
